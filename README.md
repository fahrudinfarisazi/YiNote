# YiNote - Catatan Harian Android

Aplikasi pencatatan sederhana berbasis Android yang menggunakan **Firebase Realtime Database** untuk penyimpanan data. Proyek ini dibuat untuk memenuhi tugas mata kuliah Pengembangan Aplikasi Mobile.

⚠️ **DEVELOPMENT STATUS**:
Aplikasi ini masih dalam tahap pengembangan intensif. Saat ini terdapat kendala pada sinkronisasi data yang terkadang delay dan struktur folder yang perlu dirapikan ke standar `app/src/main`. Perbaikan akan terus dilakukan seiring proses pembelajaran.

### Fitur Saat Ini:
- Simpan catatan ke Firebase.
- Tampilan list catatan menggunakan RecyclerView.
- Fitur hapus catatan.

### Tech Stack:
- Android Studio (Kotlin/Java)
- Firebase Database & Auth
- Material Design UI
